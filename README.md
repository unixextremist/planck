# planck

- a minimal git clone implementation in c

# use cases

- embedded systems
- freebsd style ports systems (ex macports haikuports xbps-src derive-ports)

# compiling

you need 

- libcurl
- libcurl-devel
- tar
- unzip (depends)
- libssl
- the internet

``gcc -o planck planck.c -lcurl``

- same for tcc
